## Initialisations

#### 1. Create server.js

```
touch server.js
```

#### 2. Initialise npm init
```
npm init --yes
```
Which automatically assign the value of "start" in package.json to server.js

#### 3. Install express
```
npm install express
```

## Usage

#### 1. Run via npm

```
npm start
```
